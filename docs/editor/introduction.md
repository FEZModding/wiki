# Introduction

FEZEditor is a multi-purpose editing program for little indie game called **FEZ**. FEZEditor is easy to use and provides tools for creating custom content such as levels, models, and various assets.

This guide contains the manual for FEZEditor. Reading this will teach you how to use the editor and its features.

## Features

- General
  - Multi-asset editing
  - Undo and Redo
  - Virtual file browser
  - Support for PAK and FEZRepacker assets
  - Mod development assistance
  - Free and cross-platform

- Level Editing
  - 3D viewport (perspective and orthographic)
  - Placement of triles, art objects, background planes, and NPCs
  - Active instances list
  - Script editing
  - Faraway preview
  - GLTF diorama export

- Trixel Editing
  - 3D model view
  - Add/remove trixels
  - Paint/bucket trixels
  - Symmetry editing
  - Emission channel editing
  - PNG import/export
  - Trile collision preview
  - Wireframe debugging preview

- Sky Editing
  - Animated time-of-day preview
  - Parameter and visual configuration
  - Shadow preview

- World Map Editing
  - Node/connection preview
  - Automatic placement
  - Vanilla tree regeneration
  - Tree structure listing

- NPC Editing (MuEditor)
  - Configure NPC behavior and sound triggers

- Music Editing (DiezEditor)
  - Create and arrange tracked music with layered loops

- Localization (PoEditor)
  - Translate game text across multiple languages

- Font Editing (ZuEditor)
  - View sprite font atlas
  - Design and edit sprite fonts with kerning

- Save Data Editing (SallyEditor)
  - Edit player progress and save slot data

## About this guide

This guide is intended to help you learn to use FEZEditor. It is not intended to teach you how to design levels, create art, or compose music - just how to use the tools to accomplish those things. Each page of this guide walks through the interface and operations you'll actually need.
